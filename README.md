# Eminem Music Page 🎤

This is my **first ever HTML project**, dedicated to Eminem — one of the greatest rappers of all time.  
The page showcases his biography, discography, top songs, fun facts, and even includes a contact form for interaction.

---

## 📄 Project Overview
- **Title:** Eminem Music Page  
- **Description:** A tribute webpage highlighting Eminem’s career, albums, and achievements.  
- **Features:**
  - Header with Eminem’s introduction
  - Navigation bar for easy section access
  - About section with a famous Eminem quote
  - Discography table with album details
  - Top songs list
  - Fun facts in a descriptive list format
  - Contact form with input validation
  - Footer with author details

---

## 🛠️ Technologies Used
- **HTML5** for structure
- Semantic elements (`header`, `nav`, `main`, `section`, `footer`)
- Forms and tables for interactive and structured content

---

## 🚀 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/eminem-music-page.git
   ```
2. Navigate into the project folder:
   ```bash
   cd eminem-music-page
   ```
3. Open the `index.html` file in your browser.
